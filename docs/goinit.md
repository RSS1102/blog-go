1. ##### 项目基本命令。

   1. 创建文件夹：

      ```
      mkdir xxx
      cd xxx
      ```
   
      
   
   2. 初始化go：
   
      ```
      go mod init xxx
      ```
   
      
   
   3. 安装`gin`：
   
      ```c
      go get -u github.com/gin-gonic/gin
      ```
   
      
   
   4. 运行项目：
   
      ```c
      go run main.go
      ```
